📐 算法口径（V6 双重约束·防前视偏差）： 股息率% = 近一年每股累计派现 ÷ 当日不复权收盘价 × 100  ·  「近一年」= 以已公告方案中最新报告期 end 倒推 1 个自然年度窗口； 生效起始日 known_from = MAX(监管最早发布日 + 15d, ex_date - 15d)，确保绝不会前置计入未公告分红。
📡 真实数据源 · 个股历史股息率
主数据源：akshare stock_zh_a_hist（东财不复权日线）+ 东财分红送股（stock_fhps_detail_em / datacenter-web 原生接口兜底）
存储 & 增量：MySQL stock_daily（按 code + date_k 主键增量补最新）· stock_dividend（按 code + ex_date + 金额/股数主键去重）； 首次请求某只个股会自动把历史全量入库；之后每天只拉「库最大日期 +1 → 今日」的缺失数据，避免频繁请求。
